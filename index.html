NS3 INSTALL and RUN On CentOS

-------------------------INSTALL-----------------------------

Open Terminal (  Click Button on Top left Corner --> System Tools --> Terminal )


cd ~/Desktop


wget https://www.nsnam.org/release/ns-allinone-3.25.tar.bz2


tar xvjf ns-allinone-3.25.tar.bz2



yum install gcc g++ python python-dev mercurial bzr gdb valgrind gsl-bin libgsl0-dev libgsl0ldbl flex bison tcpdump sqlite sqlite3 libsqlite3-dev libxml2 libxml2-dev libgtk2.0-0 libgtk2.0-dev uncrustify doxygen graphviz imagemagick texlive texlive-latex-extra texlive-generic-extra texlive-generic-recommended texinfo dia texlive texlive-latex-extra texlive-extra-utils texlive-generic-recommended texi2html python-pygraphviz python-kiwi python-pygoocanvas libgoocanvas-dev python-pygccxml


cd ns-allinone-3.25


./build.py


cd ns-3.25


--------------------------RUN --------------------

Delete all .cc files in ns-3.25/scratch  and copy first.cc from ns-3.25/examples/tutorial folder

Open Terminal ( Click Button on Top left Corner --> System Tools --> Terminal )

cd ~/Desktop


cd ns-allinone-3.25


cd ns-3.25

./waf --enable-examples --enable-tests


./waf --run scratch/first


--------------------ADDING ANIMATION AND RUN----------------

Open scratch/first.cc and add these 2 lines at top (before using namespace ns3; )


#include "ns3/netanim-module.h"

#include "ns3/mobility-module.h"


Add these lines at bottom , ( above Simulator::Run (); )


AnimationInterface anim("first.xml");

anim.AnimationInterface::SetConstantPosition(nodes.Get(0),10,25);

anim.SetConstantPosition(nodes.Get(1),40,25);


Open Terminal ( Click Button on Top left Corner --> System Tools --> Terminal )

cd ~/Desktop


cd ns-allinone-3.25


cd ns-3.25


./waf --enable-examples --enable-tests


./waf --run scratch/first


Open Terminal ( Click Button on Top left Corner --> System Tools --> Terminal )


cd ~/Desktop


cd ns-allinone-3.25


cd netanim-3.107


./NetAnim



Browse for first.xml in ns-3.25 and click on Run
